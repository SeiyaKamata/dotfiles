# Notion DWM 今日の日報を見つける・書く

## 今日の日報ページを見つける手順

### Step 1: スクリプトを実行して今日の日報URLを取得

以下のコマンドを実行する：

```
python3 "${CLAUDE_SKILL_DIR}/scripts/find_daily.py"
```

出力されたURLが今日の日報ページ。NOT_FOUNDが返った場合は自動作成がまだ動いていない可能性がある。

---

## 今日の日報を書く手順

「今日の日報を書いて」と言われた場合：

### Step 1: 上記URLの日報ページに対してnotion-fetchで現在の内容を確認する

### Step 2: チャット履歴を振り返り、以下のカテゴリで内容を整理する

- **やったこと**: 今日取り組んだタスク・作業・調査
- **気づき・学び**: 発見したこと、理解が深まったこと
- **次にやること**（あれば）: 残タスクや続きの作業

### Step 3: notion-update-pageのreplace_contentで書き込む

以下の形式で書き込む：

```
# やったこと

## [カテゴリ名]
- 内容

# 気づき・学び
- 内容
```
