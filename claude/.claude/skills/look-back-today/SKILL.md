name: look-back-today
description: 今日の作業をclaude-memで振り返り、知見をmemoryに保存する。日次振り返りを行いたいときに使う。
---

日次振り返りを行います。以下の手順で進めてください。

## 1. 今日の作業を確認

claude-memで今日の作業履歴を検索する。

```
mcp__plugin_claude-mem_mcp-search__search with dateStart=<today>, dateEnd=<tomorrow>, limit=30
```

## 2. 作業サマリーを作成

検索結果をもとに、以下の観点で整理してユーザーに提示する:

- **完了したタスク**: 何を実装・修正したか
- **未完了の作業**: コミット忘れ、テスト未実行など残っているもの
- **つまずいたポイント**: 時間がかかった箇所とその原因

## 3. 知見の抽出

ユーザーと対話しながら、明日以降に活かせる知見を洗い出す:

- 再利用できるパターンや回避策
- 今後注意すべきポイント
- ワークフローの改善点

## 4. memoryに保存

ユーザーの合意を得た上で、知見をmemoryファイルに保存する:

- 既存のmemoryファイルに追記できるものは追記
- 新しいトピックは別ファイルを作成しMEMORY.mdからリンク
- 一時的・タスク固有の情報は保存しない（安定したパターンのみ）
