# 設定ファイル集
PCを引越する際に、環境を同一にするための設定ファイル群です

設定を適応するには/binのスクリプトを実行するだけでOK


## インストールアプリ
- iTerm2
- Visual Studio Code
- Docker
- Notion (Obsidianに移行中)
- Rectangle
- WireShake
- GoogleChrome

ほかは必要になったら入れよう


- nvm[公式サイトでインストール方法の確認](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating)
